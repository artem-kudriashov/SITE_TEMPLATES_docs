# Site themes glossary

**Instant Site**\
Website builder created by Ecwid that allows creating both ecommerce and non-ecommerce sites.\
Read more about Instant Site in the [**Help Center**](https://support.ecwid.com/hc/en-us/sections/360001694100-Instant-Site).



**Ecwid admin**\
Also referred to as _Ecwid Control Panel_, it is a one-for-all browser tool, where Ecwid users can manage all aspects of their store: products, orders, customers, website, integrations, etc.&#x20;



**Instant Site editor**\
Native UI for building Ecwid-based websites from the browser. It is available to all users right from the Ecwid admin.



**Site theme**\
A pre-defined configuration for the website that includes layout, design and content settings, overall style, and in some cases custom widgets that extend the store functionality.



**Theme file**\
A `theme-name.ts` or `template-file.ts` inside the `themes` folder that lists and orders building blocks of any Ecwid website, called _sections_.



**Site section**\
A single screen-wide building block of any Ecwid website: header, footer, product browser, etc. Developers can create custom sections, then use both default and custom ones to create _Site themes_.
